## 🌟 Open Source Projects

### [Pixel Office](https://github.com/fedevgonzalez/pixel-office)
A pixel art virtual office that visualizes your Claude Code agents in real time. Watch your coding assistants work, wander, and wait as animated characters in a customizable office. Built on top of [Pixel Agents](https://github.com/pablodelucca/pixel-agents) by Pablo De Lucca.

- Real-time monitoring of multiple Claude Code sessions
- Standalone server with WebSocket dashboard
- Community layout gallery via [pixel-office-layouts](https://github.com/fedevgonzalez/pixel-office-layouts)

### [Code Orchestrator](https://github.com/fedevgonzalez/code-orchestrator)
Autonomous SaaS orchestrator powered by Claude Code. Give it a spec, get a reviewed codebase back — with multi-phase planning, auto-review & scoring, crash recovery, and real-time monitoring.

- 8 execution modes (build, fix, audit, refactor, and more)
- Auto-review with scoring (1-10) and auto-fix
- Crash recovery with checkpointing
